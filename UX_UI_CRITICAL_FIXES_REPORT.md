## Технические улучшения

### Анимации
- `fadeOut` - плавное исчезновение с коллапсом высоты
- `delayed-fadeOut` - отложенное исчезновение через 3 сек
- `slideReplace` - анимация замены контента
- `softPulse` - мягкая пульсация для привлечения внимания

### Модалки
- Единый z-index: z-[100]
- Единый фон: bg-black/85
- Единые анимации входа
- Поддержка Escape для закрытия

### Мобильная оптимизация
- Все touch targets ≥ 44px
- Правильные padding для избежания перекрытий
- Горизонтальная прокрутка для категорий
- Адаптивные кнопки
